# Инструкция: публикация статьи на Habr

## Шаг 1. Регистрация (если ещё нет аккаунта)

1. Откройте https://habr.com/
2. Нажмите "Войти" → "Регистрация"
3. Укажите email, придумайте пароль
4. Подтвердите почту через письмо
5. Заполните профиль: имя, фамилия, аватар (можно логотип Искры)

## Шаг 2. Подготовка статьи

Статья уже написана в файле `article.md`. Копируйте оттуда текст.

**Что изменить перед публикацией:**
- Уберите строку "**Автор:** Iskra Team" — Habr покажет ваш профиль автоматически
- Оставьте теги: OpenCart, PHP, e-commerce, CMS, расширения, бэкапы, миграции

## Шаг 3. Создание черновика

1. Нажмите на аватар → "Написать пост" → "Статья"
2. Вставьте заголовок: "Как мы строили экосистему для OpenCart 4: история одного магазина"
3. Вставьте текст статьи из `article.md`
4. Тип текста выберите "Публикация" (не "Новость")

## Шаг 4. Оформление

**Хабы (выберите 3-5):**
- `OpenCart` (если есть — ищите)
- `PHP`
- `E-commerce`
- `CMS`
- `Блог компании` (если публикуете от имени компании)

**Картинки (рекомендуется):**
- Загрузите скриншот лендинга: https://iskra-ecommerce.github.io/
- Скриншот админки OpenCart с модулем Iskра
- Диаграмму экосистемы (Core → Language Pack → Account)

**Как вставить картинку:**
1. Нажмите иконку картинки в редакторе
2. Загрузите файл или вставьте ссылку
3. Добавьте подпись под картинкой

## Шаг 5. Предпросмотр и публикация

1. Нажмите "Предпросмотр" — проверьте, что таблицы отображаются корректно
2. Нажмите "Опубликовать"
3. Выберите "В черновик" → потом "На модерацию" (или сразу "На модерацию")
4. Статья уйдёт на проверку (обычно 1-24 часа)

## Шаг 6. После публикации

- Добавьте ссылку на статью в README репозиториев
- Разошлите в Telegram-каналы
- Поделитесь в соцсетях

---

**Важно:** На Habr нельзя прямую рекламу. Статья должна быть полезной (как у нас — рассказ о проблемах и решениях). Не пишите "Купите наш модуль" — пишите "Вот как мы решили проблему".
