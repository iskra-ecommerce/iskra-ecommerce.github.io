# Чеклист модерации Habr

## Обложка (КДПВ)
- [ ] Размер: 780 × 440 пикселей
- [ ] Формат: PNG, < 1 МБ
- [ ] Нет GIF, нет вертикальной ориентации
- [ ] Нет грубого/эротического содержания
- [ ] Контрастная, читаемая в миниатюре

## Статья
- [ ] IT-тематика (OpenCart, PHP, e-commerce)
- [ ] Авторский контент, не копипаста
- [ ] Нет прямой рекламы ("Купите наш модуль" → "Вот как мы решили проблему")
- [ ] Нет кричащих заголовков ("Шок! Вирусы!")
- [ ] Разбита на абзацы, хорошо отформатирована
- [ ] Код оформлен блоками с подсветкой синтаксиса
- [ ] Ссылки встроены в слова, не голые URL
- [ ] Грамотный русский язык, без "падонкаффского" сленга
- [ ] Картинки залиты на Habrastorage
