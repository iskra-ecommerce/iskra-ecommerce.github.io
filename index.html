Как всё началось

Представьте: три ночи, остыл кофе, клиент звонит — «сайт упал, база сломалась, бэкапа нет». Разработчик сидит, смотрит в экран и думает: «Почему это никто не сделал нормально?»

Так родилась Искра. Не модуль, не плагин — а экосистема. Три расширения, которые живут друг с другом, как соседи в старом дворе: кто-то держит лестницу, кто-то — ведро, кто-то — краску. Вместе красят забор, по отдельности — бесполезны.

Всё под лицензией MIT. Бесплатно. Как воздух.

Что умеет

Всё, что нужно для магазина на OpenCart 4 в СНГ

📋

Дневник капитана

Логи, где видно всё: кто лез, куда лез, что сломал. Уровни от DEBUG до CRITICAL. Автоочистка — как дворник.

🗄️

Грузчики с планом

Миграции БД с rollback. Сказал «стол сюда» — стол здесь. Контрольные суммы, версии, журнал учёта.

💾

Страховка, которая не моргает

Бэкапы делаются сами. SQL-дампы хранятся, сортируются, восстанавливаются одной кнопкой. Не надо лезть в консоль.

🔍

Доктор перед операцией

Проверка целостности таблиц перед миграциями. Если что-то не так — стоп, разбираемся. Не лезем вслепую.

🌐

Пять языков, одна установка

RU, UK, KK, BE, RO — нативные переводы, не машинные. Человек думал, как сказать по-казахски «добавить в корзину».

🔐

Регистрация, которая не бесит

Валидация в реальном времени, маски телефонов, индикатор пароля. Всё на пяти языках — выбрал в шапке, форма переключилась.

Быстрый старт

Установка за 2 минуты через Extension Installer

1. Скачайте .ocmod.zip из GitHub Releases
2. System → Extensions → Extension Installer → Upload
3. Extensions → Modules → Install → Enable
4. Идите спать — оно работает
