Как всё началось

Представьте: три ночи, остыл кофе, клиент звонит — «сайт упал, база сломалась, бэкапа нет». Разработчик сидит, смотрит в экран и думает: «Почему это никто не сделал нормально?»

Так родилась Искра. Не модуль, не плагин — а экосистема. Три расширения, которые живут друг с другом, как соседи в старом дворе: кто-то держит лестницу, кто-то — ведро, кто-то — краску. Вместе красят забор, по отдельности — бесполезны.

Всё под лицензией MIT. Бесплатно. Как воздух.

Быстрый старт

Установка за 2 минуты через Extension Installer

Скачайте архив

Перейдите на GitHub Releases и скачайте .ocmod.zip для нужного модуля.

Загрузите в админку

Откройте System → Extensions → Extension Installer и загрузите .ocmod.zip.

Активируйте модуль

Перейдите в Extensions → Modules, найдите Iskra, нажмите Install и Enable.

1. Скачайте .ocmod.zip из GitHub Releases
2. System → Extensions → Extension Installer → Upload
3. Extensions → Modules → Install → Enable
4. Идите спать — оно работает

Часто задаваемые вопросы

Отвечаем на то, что спрашивают чаще всего

Как установить Iskra Core на OpenCart 4?

Скачайте .ocmod.zip из GitHub Releases, загрузите через System → Extensions → Extension Installer, обновите кэш модификаторов. Модуль готов к работе.

Поддерживает ли Iskra кириллицу и казахский язык?

Да. Полная поддержка UTF-8 mb4. Language Pack включает русский, украинский, казахский, белорусский и румынский языки — нативные переводы, не машинные.

Можно ли использовать расширения бесплатно в коммерческих проектах?

Да. Все расширения распространяются под лицензией MIT. Можно использовать в личных и коммерческих проектах без ограничений.

Как работает автоматическое логирование?

Iskra Core ведёт централизованный журнал событий с уровнями DEBUG, INFO, WARNING, ERROR, CRITICAL. Поддерживает фильтрацию, категории и автоочистку по размеру таблицы или количеству строк.

Есть ли поддержка миграций базы данных?

Да. Встроенная система миграций с отслеживанием версий в таблице iskra_migration. Каждая миграция имеет up и down методы, контрольную сумму и статус. Rollback поддерживается.
